# Trans*py*ling

A tour of web transpiling and back again with five different languages.

Prepared for the annual :snake: Fuzzy.py :snake: extravaganza.

## Conversions

- `Python` (with PythonJS) --> `JS`
- `Ruby` (with Opal) --> `JS`
- `JS` (with js2php) --> `PHP`
- `JS` (with js2py) --> `Python`
- `JS` (with js2hd) --> `Hodor`

BONUS

- `Dogescript` (with dogescript) --> `JS`
